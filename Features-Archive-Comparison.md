# Archive Comparison

Compare multiple prints side-by-side to identify differences in settings, materials, and outcomes. Perfect for troubleshooting failures or optimizing print settings.

## Overview

- **Side-by-side comparison** — View 2-5 archives at once
- **Difference highlighting** — Settings that differ are highlighted
- **Success correlation** — Compare successful vs failed prints
- **Export insights** — Save comparison data

## Using Archive Comparison

### Selecting Archives

1. Navigate to the **Archives** page
2. Click the **Select** button (or press `S`)
3. Click on 2-5 archives to compare
4. Click the **Compare** button that appears

### Alternative Selection

- Right-click an archive → **Select**
- Use Shift+Click for range selection
- Use Ctrl/Cmd+Click for individual selection

## Comparison View

The comparison modal shows a table with:

| Column | Description |
|--------|-------------|
| Setting | The setting name |
| Archive 1 | Value from first archive |
| Archive 2 | Value from second archive |
| ... | Additional archives |

### Highlighting

- **Yellow background** — Values that differ between archives
- **No highlight** — Values that are the same

## Compared Settings

The comparison includes:

### Print Settings
- Layer height
- Infill percentage
- Print speed
- Temperature settings
- Support settings

### Material
- Filament type
- Filament brand
- Color

### Results
- Print status (completed/failed)
- Actual print time
- Filament used

### Metadata
- Printer used
- Print date
- Designer

## Use Cases

### Troubleshooting Failures

1. Select a failed print and a successful print of the same model
2. Compare settings to identify differences
3. Look for patterns (temperature, speed, support)

### Optimizing Settings

1. Print the same model with different settings
2. Compare results to find optimal configuration
3. Note which changes improved quality/speed

### Comparing Printers

1. Print the same model on different printers
2. Compare accuracy and print times
3. Identify printer-specific adjustments needed

## Closing the Modal

- Click the **Close** button
- Click the **X** in the corner
- Press **Escape**
- Click outside the modal (backdrop)

## Tips

- Compare prints of the same model for best insights
- Include both successful and failed prints
- Document findings in archive notes
- Use comparison data to create optimal presets
